# Network Scanning Mobile Application

## Proje Amacı
Bu proje, React Native kullanılarak geliştirilecek bir mobil ağ tarama uygulamasıdır. 
Uygulama, yerel ağdaki aktif cihazları tespit etmeyi ve bu verileri bir API aracılığıyla veritabanına kaydetmeyi amaçlamaktadır.

## Kullanılacak Teknolojiler
- React Native (Expo)
- Node.js (Backend)
- Express.js (API)
- MongoDB / MySQL (Veritabanı)

## Planlanan Özellikler
- Yerel ağ tarama
- Cihaz listesi görüntüleme
- Tarama geçmişi kaydetme
- API entegrasyonu
