# Network Scanning Mobile Application

## Proje Amacı
Bu proje, React Native kullanılarak geliştirilecek bir mobil ağ tarama uygulamasıdır. 
Uygulama, yerel ağdaki aktif cihazları tespit etmeyi ve bu verileri bir API aracılığıyla veritabanına kaydetmeyi amaçlamaktadır.

## Kullanılacak Teknolojiler
- React Native (Expo)
- Node.js (Backend)
- Express.js (API)
- MongoDB / MySQL (Veritabanı)

## Planlanan Özellikler
- Yerel ağ tarama
- Cihaz listesi görüntüleme
- Tarama geçmişi kaydetme
- API entegrasyonu

## 1. Hafta Çalışmaları
- Proje konusu belirlendi
- Gerekli teknolojiler araştırıldı
- React Native projesi oluşturuldu
- İlk kullanıcı arayüzü başlatıldı

## 2. Hafta Çalışmaları

- Uygulama Expo Router kullanılarak çok sayfalı hale getirildi  
- Ana ekran (Home), tarama ekranı (Scan) ve geçmiş ekranı (History) oluşturuldu  
- Sayfalar arası geçiş (navigation) yapısı kuruldu  
- Ağ tarama işlemi için örnek (fake) veri ile çalışan bir simülasyon geliştirildi  
- Kullanıcı arayüzü düzenlenerek daha modern ve kullanıcı dostu hale getirildi
